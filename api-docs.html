NoteTaker API Documentation

General Notes

All endpoints return JSON responses. Dates should be in ISO format (YYYY-MM-DD).

Todo Endpoints

List Todos for Date

GET /api/todos/date/:date
date - Date in YYYY-MM-DD format

Returns an array of todos for the specified date.

Create Todo for Date

POST /api/todos/date/:date
date - Date in YYYY-MM-DD format
Body - JSON object with:
  • content (string) - Todo content

Toggle Todo Completion

PUT /api/todos/:id/toggle
id - Todo ID

Delete Todo

DELETE /api/todos/:id
id - Todo ID

Event Endpoints

List Events for Date

GET /api/events/date/:date
date - Date in YYYY-MM-DD format

Create Event for Date

POST /api/events/date/:date
date - Date in YYYY-MM-DD format
Body - JSON object with:
  • content (string) - Event content

Delete Event

DELETE /api/events/:id
id - Event ID

Reminder Endpoints

List Reminders for Date

GET /api/reminders/date/:date
date - Date in YYYY-MM-DD format

Create Reminder

POST /api/reminders/date/:date
date - Date in YYYY-MM-DD format
Body - JSON object with:
  • content (string) - Reminder content
  • time (string) - Time in HH:mm format

Delete Reminder

DELETE /api/reminders/:id
id - Reminder ID

General Endpoints

Get All Notes

GET /api/notes
type (optional) - Filter by note type
date (optional) - Filter by date

Get Calendar Events

GET /api/calendar
start (optional) - Start date
end (optional) - End date

Execute Python Script

POST /api/execute-python
Body - JSON object with:
  • text (string) - The text to process with the Python script

Returns the output from the Python script.
