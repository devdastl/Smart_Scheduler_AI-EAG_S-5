NoteTaker API Documentation

This document provides information about the available API endpoints for the NoteTaker application.

Base URL

All API endpoints are relative to the base URL: http://localhost:3000

Authentication

Currently, the API does not require authentication.

Endpoints

GET /api/notes

Get all notes with optional filtering.

Query Parameters

type (string)

Filter notes by type (todo, event, blocker, reminder).

date (string)

Filter notes by date (YYYY-MM-DD format).

Response

Returns an array of note objects.

Example

GET /api/notes?type=todo&date=2023-05-15 Response: [ { "id": "123456", "type": "todo", "content": "Complete project", "date": "2023-05-15", "completed": false, "createdAt": "2023-05-14T12:00:00.000Z" } ]

GET /api/notes/:id

Get a specific note by ID.

URL Parameters

id (string) Required

The ID of the note to retrieve.

Response

Returns a note object.

Example

GET /api/notes/123456 Response: { "id": "123456", "type": "todo", "content": "Complete project", "date": "2023-05-15", "completed": false, "createdAt": "2023-05-14T12:00:00.000Z" }

POST /api/todos

Create a new todo.

Request Body

content (string) Required

The content of the todo.

date (string) Required

The date of the todo (YYYY-MM-DD format).

Response

Returns the created todo object.

Example

POST /api/todos Content-Type: application/json { "content": "Complete project", "date": "2023-05-15" } Response: { "id": "123456", "type": "todo", "content": "Complete project", "date": "2023-05-15", "completed": false, "createdAt": "2023-05-14T12:00:00.000Z" }

POST /api/events

Create a new event.

Request Body

content (string) Required

The content of the event.

date (string) Required

The date of the event (YYYY-MM-DD format).

Response

Returns the created event object.

POST /api/blockers

Create a new blocker.

Request Body

content (string) Required

The content of the blocker.

date (string) Required

The date of the blocker (YYYY-MM-DD format).

startTime (string)

The start time of the blocker (HH:MM format).

endTime (string)

The end time of the blocker (HH:MM format).

Response

Returns the created blocker object.

POST /api/reminders

Create a new reminder.

Request Body

content (string) Required

The content of the reminder.

date (string) Required

The date of the reminder (YYYY-MM-DD format).

time (string)

The time of the reminder (HH:MM format).

Response

Returns the created reminder object.

PUT /api/notes/:id

Update a note.

URL Parameters

id (string) Required

The ID of the note to update.

Request Body

content (string)

The updated content of the note.

date (string)

The updated date of the note (YYYY-MM-DD format).

completed (boolean)

Whether the note is completed (for todos).

time (string)

The updated time of the note (HH:MM format, for reminders).

startTime (string)

The updated start time of the note (HH:MM format, for blockers).

endTime (string)

The updated end time of the note (HH:MM format, for blockers).

Response

Returns the updated note object.

DELETE /api/notes/:id

Delete a note.

URL Parameters

id (string) Required

The ID of the note to delete.

Response

Returns a success object.

Example

DELETE /api/notes/123456 Response: { "success": true }

GET /api/calendar

Get calendar events.

Query Parameters

start (string)

The start date (ISO format).

end (string)

The end date (ISO format).

Response

Returns an array of calendar event objects.

GET /api/input

Get input text.

Response

Returns an object with the input text.

Example

GET /api/input Response: { "text": "Input text" }

POST /api/input

Set input text.

Request Body

text (string) Required

The text to set as input.

Response

Returns an object with the set text.

Example

POST /api/input Content-Type: application/json { "text": "New input text" } Response: { "text": "New input text" }

GET /api/output

Get output text.

Response

Returns an object with the output text.

POST /api/output

Set output text.

Request Body

text (string) Required

The text to set as output.

Response

Returns an object with the set text.
